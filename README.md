Mall Customer Segmentation using K-Means

Objective
To segment customers based on their annual income and spending score using K-Means clustering.

Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib

Dataset
Mall Customers dataset containing:
- Age
- Annual Income (k$)
- Spending Score (1-100)

Steps Performed
1. Loaded the dataset
2. Selected relevant features
3. Applied feature scaling using StandardScaler
4. Used K-Means algorithm for clustering
5. Visualized clusters using 2D and 3D plots

Output
Customers are grouped into different clusters based on similar spending behavior.

 Visualizations
- 2D Cluster Plot
- 3D Cluster Plot

 Conclusion
K-Means clustering helps in identifying different types of customers, which can be useful for targeted marketing.
